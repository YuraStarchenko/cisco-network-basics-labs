# Lab 02 — Home Network Setup

## Packet Tracer – Connecting to a Web Server

## Goals and Tasks
Observe how packets are sent over the Internet using IP addresses.

## Part 1: Test Connection to the Web Server
a. Open the command line on the source host. Select PC0.
b. Go to Desktop → Command Prompt.
c. Test the connection to the web server.
Type the command: ping 172.33.100.50

Testing connection to the web server using ping.

![Home Network Setup](/lab02-home-network/screenshots/Screenshot1.png)

The reply shows that the client is connected to the destination web server.
The first reply may take some time because devices are loading and ARP is working.

## Part 2: Connect to the Web Server Using a Web Client

a. On PC0 → Desktop, open Web Browser.
b. Enter 172.33.100.50 in the URL bar and click Go.
The web page opens using the server IP address.

Testing connection to the web server using Web Browser.
